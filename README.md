# DFCData

Export of fleet data from the Dropfleet Commander PDFs.

See the [system](system_data_format.md), [fleet](fleet_data_format.md), and [civilian](civilian_data_format.md) data format descriptions for details of the structure of the data files.

Formal schemas are located in the `Schemas` directory.

The [master index](Data/_index.json) is the entry point for consuming the data.

<br/>

---

<br/>

The data here is extracted from the Dropfleet Commander PDFs linked from the [Dropfleet Commander Resources](https://ttcombat.com/pages/dropfleet-commander-downloads) page. These data files contain only the publicly available information provided in those PDFs. Neither the resources page nor the individual PDFs provide copyright information and therefore I assume that copyright belongs to TTCombat. I do not claim any ownership of any content extracted from the PDFs. This repository and the data within are not endorsed by TTCombat.

This data is provided for the purpose of easing the creation and maintainance of tools for the Dropfleet Commander game. No other purpose is intended or supported.
